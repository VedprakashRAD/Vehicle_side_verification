About This System

This vehicle verification system uses advanced AI to:

  • Verify vehicle orientation using an ensemble model trained on thousands of vehicle images
  • Detect and extract license plate information using optical character recognition (OCR)
  • Ensure the uploaded images match the expected vehicle orientation
  • Verify towing vehicles and their associated license plates

The system uses an ensemble model with 90.5% accuracy for orientation detection.
